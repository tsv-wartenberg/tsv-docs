# tsv-dashboard — Zentrales Dashboard

## Uebersicht

Eigenstaendige React-App, die alle TSV-Apps in einer zentralen Oberflaeche aggregiert.

| Feld | Wert |
|------|------|
| **Repo** | Noch nicht erstellt |
| **Technologie** | React + TypeScript (geplant) |
| **Status** | Geplant |

## Konzept

- Widgets/Karten fuer jede App (tsv-hub Stats, HelferHub Schichten, etc.)
- Links zu den einzelnen Apps
- Zentrale Navigation
- Authentifizierung ueber tsv-auth

## Herkunft

Kann aus dem bestehenden `Dashboard.tsx` in tsv-hub extrahiert werden.
