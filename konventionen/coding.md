# Coding-Konventionen

## Python (Backend)

- **Version**: Python 3.12+
- **Framework**: FastAPI 0.115+
- **ORM**: SQLAlchemy 2.0+ (async)
- **Schemas**: Pydantic v2
- **Linting**: ruff
- **Formatting**: ruff format
- **Tests**: pytest + pytest-asyncio + httpx
- **Type Hints**: Ueberall
- **Async/Await**: Fuer alle DB-Operationen

## TypeScript (Frontend)

- **Framework**: React 19+ mit TypeScript (strict)
- **Build Tool**: Vite
- **UI**: Tailwind CSS + shadcn/ui
- **State**: TanStack Query (Server State)
- **Formulare**: react-hook-form + zod
- **Routing**: React Router v6+
- **Linting**: ESLint + Prettier
- **Keine** `any` Typen

## Allgemein

- **UI-Texte**: Deutsch
- **Code**: Englisch (Variablen, Funktionen, Klassen)
- **Kommentare**: Deutsch erlaubt, Englisch bevorzugt
- **Dokumentation**: Deutsch (da Vereinskontext)
